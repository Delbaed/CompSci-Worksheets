week1: finding dataset

First I chose the World Spider Database, but upon attempting to sift through data, I found it was a very small and restrictive set. Instead I opted for the Weather Bouy Network, which has more columns of variables which sparked better ideas for graphs.

week2: cleaning dataset

I used the Pandas library to clean my dataset. First I cut down my dataset for ease of plotting. I struggled to create a list of whtb I wanted to remove, before settling on a for loop to create a list of the rows I no longer needed. Then I removed unessecary columns with empty or unneeded data. I removed the columns "station_id, "longitude", "latitude", "DewPoint" and "QC_Flag".

week3: scatter plot

I initially tried to plot a scatter plot in week2, before cutting down my dataset, but the dataset was so big(over eight hundred thousand rows) that my laptop refused to display the plot. So after week2's work I made my scatter plot again, to find any correlation between "AirTemperature" and "SeaTemperature".

week4: bar chart & linear plot

I tried to create a bar chart, however I could not figure out how to plot it the way I wanted. So I opted to do a linear plot of wind-speeds against time instead. This had its own challenges as I wanted to create filters based on the station the data was collected from.
